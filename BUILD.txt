How to build equal5?
====================

To build equal5 you will need gradle build system and JDK.
Follow this instructions:
1. Download JDK(version 1.6 is OK) at http://www.oracle.com/technetwork/java/javase/downloads/index.html
2. Unpack/install
3. Put JAVA_HOME and modify PATH system variables
4. Download last Gradle(now I use 1.5) http://www.gradle.org/downloads
5. Unpack
6. Put GRADLE_HOME and modify your PATH system variables
(or you'll need to run with absolute path)
7. Run "gradle clean build"

Thank you!
